# 🚀 Hero IO - Mobile App Directory

**Hero IO** হলো একটি আধুনিক ওয়েব প্ল্যাটফর্ম যেখানে বিভিন্ন ক্যাটাগরির মোবাইল অ্যাপ্লিকেশনের (APK) সংগ্রহ রাখা এবং প্রদর্শন করা হয়। এটি ডেভেলপার এবং ইউজারদের জন্য একটি সাজানো অ্যাপ ডিরেক্টরি হিসেবে কাজ করে।

## ✨ বৈশিষ্ট্যসমূহ (Features)

- 📱 **অ্যাপ গ্যালারি:** সব ধরনের মোবাইল অ্যাপের সুন্দর প্রদর্শন।
- 🔍 **স্মার্ট সার্চ:** নাম দিয়ে সহজেই প্রিয় অ্যাপ খুঁজে পাওয়ার সুবিধা।
- 📊 **রেটিং ও রিভিউ:** অ্যাপের পারফরম্যান্স বোঝার জন্য রেটিং সিস্টেম।
- ⚡ **ফাস্ট নেভিগেশন:** রিঅ্যাক্ট রাউটার ব্যবহারের ফলে খুব দ্রুত এক পেজ থেকে অন্য পেজে যাওয়া যায়।
- 🎨 **আধুনিক ডিজাইন:** Tailwind CSS এবং daisyUI দিয়ে তৈরি আই-ক্যাচি ইন্টারফেস।

## 🛠️ ব্যবহৃত প্রযুক্তি (Technologies Used)

প্রজেক্টটি তৈরিতে আধুনিক সব ওয়েব টুলস ব্যবহার করা হয়েছে:

- **Frontend Framework:** [React.js](https://reactjs.org/) (Vite)
- **Routing:** [React Router](https://reactrouter.com/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [daisyUI](https://daisyui.com/)
- **Icons:** React Icons / Lucide React
